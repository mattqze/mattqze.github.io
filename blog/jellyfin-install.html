What is Jellyfin?

Jellyfin is a media server, like Plex Media Server, it hosts your content (usually movies, series, music and photos) and lets you access your content from any interface, from TVs to Phones. Unlike Plex. Jellyfin is completely open source, meaning all of the code that runs the apps and server is publicly available and can be modified to your tastes. This does not make Jellyfin insecure, instead, Jellyfin becomes more security hardened with the help of the community surrounding Jellyfin and it's apps. Since Jellyfin runs on your local network, no data is used. Making it perfect for limited connections.

How to use Jellyfin.

There are many ways to set up and use Jellyfin. I opted for Docker, due to its ease of use and seperation from my main system. However you can install Jellyfin on Windows, Mac and Linux. Below is the command to get you started with using Jellyfin on Docker. (You must have some form of content to use Jellyfin, I will not cover ways to obtain that content here.)


Docker Compose:

version: "3"
services:
  jellyfin:
    image: lscr.io/linuxserver/jellyfin
    container_name: jellyfin
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=America/Miami
    volumes:
      - ./config:/config
      - /path/to/media:/mediatype
    ports:
      - 8096:8096
    restart: unless-stopped
docker-compose up -d

You can download the Docker Compose file here.


From there, Jellyfin is quite easy to set up. All you need to do after set up is download the Jellyfin apps and enjoy your content.
